# Dashboard_GumovskiMira_SupplementaryMaterial
This repository contains an interactive Streamlit dashboard for exploring the Gumovski-Mira map. Users can visualize and interact with attractors, bifurcation diagrams, and the parameter space. The dashboard is supplementary material for the paper "Parameter Space for Gumovski-Mira Map" by Diego Fregolent Mendes de Oliveira and Daniel Borin.
